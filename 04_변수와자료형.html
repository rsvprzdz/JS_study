변수와 자료형

변수 선언시 var, let, const라는 키워드를 이용하여 선언 가능

* 변수 선언 방법

* 자료형

변수 선언 시 자료형을 별도로 지정하지 않음 (=> 자료형의 개념이 없는 것은 아님)
=> 한줄씩 실행되는 인터프리터 언어 특성상 변수에 대입되는 값에 따라서 알아서 자료형이 결정됨

자바스크립트의 자료형

  • 숫자 : Number
  • 문자열 : String
  • 논리값 : Boolean
  • undefined
  • null
  • Object

* 변수 명명 규칙

  1. 일반적으로 영어를 사용하고, 문자와 숫자 모두 사용
  2. 특수문자 _ 또는 $만 사용 가능
  3. 숫자로 시작할 수 없음 ex) 100_
  4. 키워드(예약어)를 변수명으로 사용할 수 없음 ex) let, const, ...

표기법 (Naming Type)

  1. camelCase : 대부분 사용되는 표기법. 소문자로 시작하고 단어의 시작마다 대문자로 구분.
  2. snake_case : 파이썬에서 주로 사용되는 표기법.단어의 시작마다 _로 구분.
  3. PascalCase : c# 또는 ms계열 언어에서 사용되는 표기법. 모든 단어의 시작을 대문자로 표기
