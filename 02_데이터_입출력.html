1. 데이터를 출력하는 구문

  1. [window.]alert("알림창에 출력하고자 하는 내용");
  2. [window.]console.log("콘솔창에 출력하고자 하는 내용");
  3. document.write("화면 상에 출력하고자 하는 내용")
  4. 선택한요소.innerHTML | .innerText = "선택된 요소에 출력할 내용"

2. 데이터를 입력받는 구문 (변수에 저장/기록)

  1. 변수 = [window.]confirm("질문 내용");
  2. 변수 = [window.]prompt("질문 내용");
  3. 변수 = 선택한요소.속성; (id, className, innerText, ...)
  4. 변수 = 선택한input요소.value;

[window.]confirm("질문내용");

confirm 호출 시 "질문내용", 확인/취소 버튼이 존재하는 알림창이 표시됨 확인 버튼 클릭 시 true, 취소 버튼 클릭 시 false를 반환


[window.]prompt("질문 내용")

prompt 호출 시 "질문 내용"과 입력할 수 있는 "텍스트 상자", 확인/취소 버튼이 보여지는 알림창이 표시됨 확인/취소 버튼이 보여지는 알림창이 표시됨
확인버튼 클릭 시 텍스트 상자에 입력된 값을 문자열로 반환 취소 버튼 클릭 시 null을 반환 (값이 입력되어 있어도 취소버튼 클릭 시 null 반환)
